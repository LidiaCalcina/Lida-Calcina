TEJIDO LISO

Es la técnica de tejido más simple y se utiliza para producir telas livianas y casi transparentes, como la muselina y el césped de algodón. La tela se produce pasando el hilo de trama por debajo de un hilo de urdimbre, luego sobre el siguiente hilo de urdimbre, y así sucesivamente.

 

Tejido liso

 

 

Esta técnica produce telas resistentes y duraderas con un patrón diagonal que se produce variando la forma en que el hilo de trama pasa por debajo o por encima del hilo de urdimbre.

Tejido de satén

 

 

Es similar al tejido de sarga, excepto que no tiene un patrón diagonal. Ambas técnicas producen telas resistentes y duraderas.

 

Tejido de tapices

Esta técnica se basa en hilos de trama de una variedad de colores para producir imágenes detalladas. Para hacer un tapiz, una persona generalmente coloca una imagen detrás de los hilos de urdimbre estirados y teje la trama para que coincida con la imagen.

La textilería es el arte y la técnica de fabricar telas y tejidos. Aquí te presento dos conceptos relacionados con la textilería:

Tejido: Es el proceso mediante el cual se entrelazan los hilos para formar una tela. El tejido puede ser realizado a mano o con ayuda de una máquina, y existen diferentes técnicas para lograr distintos patrones y texturas

Hilandería: Es el proceso de convertir las fibras en hilos para fabricar telas. Este proceso se realiza mediante el uso de una rueca o un huso, que se encarga de estirar y retorcer las fibras para formar hilos resistentes y uniformes..
